# Usage
<!-- usage -->
```sh-session
$ npm install -g wcmd
$ wcmd COMMAND
running command...
$ wcmd (-v|--version|version)
wcmd/1.0.0 darwin-x64 node-v12.13.1
$ wcmd --help [COMMAND]
USAGE
  $ wcmd COMMAND
...
```
<!-- usagestop -->

`Note` the `-l` and `-m` options cannot be used at the same time.
